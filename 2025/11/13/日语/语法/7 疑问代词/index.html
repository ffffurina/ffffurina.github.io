Lesson 7: 全能询问系统 —— 5W1H 的“原位替换”

在英语中，提问通常需要把 What / Where 提到句首（Move Alpha）。但在日语中，逻辑非常简单：哪里不懂改哪里。

不需要移动词序，只需要把你想问的那个部分（主语、宾语、地点、时间）替换成对应的“疑问词”即可。

1. 核心疑问词清单 (Query Variables)

请务必背熟这几个“变量名”：

汉字 (假名) 读音 含义 备注
何 (なに / なん) nani / nan 什么 后面接 d/t/n 开头的音时读 nan
誰 (だれ) dare 谁 礼貌说法是 どなた (donata)
どれ dore 哪个 which
いつ itsu 什么时候 后面通常不加 に
どこ doko 哪里 场所
どう dou 怎么样 询问状态/意见
いくら ikura 多少钱 价格

2. 难点攻克：何 (Nani) vs 何 (Nan)

这是初学者最容易晕的地方。虽然汉字都是“何”，但读音会根据后面的音节发生“连读变音”。

  • 读 Nan (なん) 的情况：
    1. 后面跟着 desu：何 (なん) ですか。 (Nan desu ka / 是什么？)
    2. 后面跟着 t / d / n 行的助词：
      • 何 (なん) の (Nan no / 什么的…)
      • 何 (なん) で (Nan de / 用什么…)
  • 读 Nani (なに) 的情况：
    1. 后面跟着 o (を) 或 ga (が)：
      • 何 (なに) を食 (た) べますか。 (Nani o tabemasu ka / 吃什么？)
      • 何 (なに) がありますか。 (Nani ga arimasu ka / 有什么？)

3. 句型实战 (Implementation)

我们用之前的句型来演示“原位替换”法。

场景 A：询问时间 (Time)
  • 陈述：明日 (あした) 行 (い) きます。 (明天去。)
  • 提问：いつ 行 (い) きますか。 (什么时候去？)
    • Code: Time 替换为 いつ。
    • 注意：いつ 后面不要加 に。
场景 B：询问手段 (Method)
  • 陈述：電車 (でんしゃ) で行 (い) きます。 (坐电车去。)
  • 提问：何 (なん) で行 (い) きますか。 (怎么去？/ 用什么去？)
    • Code: Tool 替换为 何。注意读音是 Nan。
场景 C：询问感想 (Opinion/State)

这是一个非常实用的句型，用于询问对方对某事的看法。

  • 公式：Topic + は + どうですか。
  • 例句：
    • 日本 (にほん) の生活 (せいかつ) はどうですか。
      (Nihon no seikatsu wa dou desu ka?)
      (日本的生活怎么样？)
    • 回答：楽 (たの) しいです。 (Tanoshii desu / 很开心。)

4. 词汇穿插：常用的“后缀”询问

利用汉字组合，我们可以构造出更多疑问词。

组合 读音 含义 例子
何 (なん) + 曜日 (ようび) nan-youbi 星期几 今日 (きょう) は何曜日 (なんようび) ですか。
何 (なん) + 時 (じ) nan-ji 几点 今 (いま) 、何時 (なんじ) ですか。
何 (なん) + 歳 (さい) nan-sai 几岁 何歳 (なんさい) ですか。

5. 深度逻辑：疑问词做主语 (Subject Focus)

还记得 wa (は) 和 ga (が) 的区别吗？这里有一个铁律：

Rule: 如果疑问词（谁、哪个、什么）在句子里做主语，助词必须用 が (ga)，绝对不能用 は。回答时也要用 が。

  • 错误 (Bug)：~~誰 (だれ) は 来 (き) ますか。~~
  • 正确 (Pass)：誰 (だれ) が 来 (き) ますか。 (Who is coming?)
    • 回答：田中 (たなか) さん が 来 (き) ます。 (田中来。)
