Lesson 13: 动词的名词化封装 —— 字典形 (Dictionary Form) 与能力表达

我们之前一直在学怎么把动词变来变去（Masu, Te, Nai）。今天我们要回到原点——字典形 (Dictionary Form / 辞書形)。
也就是动词最原始、未经过任何修饰的状态（你在字典里查到的那个样子）。

但这节课的重点不是“形态”，而是“封装” (Encapsulation)。

1. 核心概念：动词的名词化 (Nominalization)

在日语中，动词（比如“吃”）不能直接做主语或宾语。你不能说“我喜欢吃（I like eat）”，必须说“我喜欢吃这件事（I like eating）”。

我们需要一个“包装器” (Wrapper) 把动词转换成名词。
这个包装器就是：こと (koto)。

  • 公式：
    • Logic: Function $\rightarrow$ Object / Event
  • 例子：
    • 食 (た) べる (吃) $\rightarrow$ 食 (た) べること (吃这件事 / Eating)
    • 見 (み) る (看) $\rightarrow$ 見 (み) ること (看这件事 / Watching)

2. 语法点 A: 表达能力 (Can do)

要表达“我会做某事 / 我能做某事”，我们使用以下结构：

  • 直译：关于做某事这件事，是可行的 (is possible)。

  • 词汇: できます (dekimasu) = 能/可以 (Can/Able to)。
    否定：できません
    实战演练 (Implementation)：

  • 场景：我会说日语。

    • Verb: 話 (はな) す (hanasu / 说)
    • Wrapper: 話 (はな) すこと (Speaking)
    • Sentence: 私 (わたし) は 日本語 (にほんご) を 話 (はな) すこと が できます。
    • (Watashi wa Nihongo o hanasu koto ga dekimasu.)

  • 场景：我不会游泳。

    • Verb: 泳 (およ) ぐ (oyogu / 游泳)
    • Sentence: 私 (わたし) は 泳 (およ) ぐこと が できません。
    • (Watashi wa oyogu koto ga dekimasen.)

简化版：如果是“会钢琴”、“会滑雪”这种名词，直接说 ピアノができます 即可。只有动词才需要 koto 封装。

3. 语法点 B: 表达兴趣 (Hobbies)

要介绍“我的爱好是…”，动词也必须名词化。

实战演练：

  • 场景：我的爱好是看电影。
    • Verb: 見 (み) る
    • Sentence: 私 (わたし) の 趣味 (しゅみ) は 映画 (えいが) を 見 (み) ること です。
    • (Watashi no shumi wa eiga o miru koto desu.)
      「映画を見る:看电影」

      4. 语法点 C: 时间顺序 —— 在…之前 (Before)

字典形 的另一个重要用途是表达时间序列。因为字典形含有“未完成/将来”的语义，所以用来表示“在做 X 之前”。

  • 注意：无论整个句子是过去时还是将来时，前に 前面的动词永远用原形（因为相对于动作发生时，它还没发生）。

实战演练：

  • 场景：每天睡觉前，刷牙。
    • Action 1 (Before): 寝 (ね) る (neru)
    • Action 2 (Main): 歯 (は) を 磨 (みが) きます (ha o migakimasu / brush teeth)
    • Sentence: 寝 (ね) る 前 (まえ) に、歯 (は) を 磨 (みが) きます。
      「歯を磨く：刷牙」
  • 场景：来日本之前，我学习了日语。
    • Action 1 (Before): 日本 (にほん) へ 来 (く) る (kuru)
    • Action 2 (Main / Past): 勉強 (べんきょう) しました (benkyou shimashita)
    • Sentence: 日本 (にほん) へ 来 (く) る 前 (まえ) に、日本語 (にほんご) を 勉強 (べんきょう) しました。
    • (注意：虽然是过去的动作，但前面依然用 来る，不用 来た)

🛠️ 词汇穿插：技能与活动

✅ 课后小练习 (Output)

请利用 字典形 + こと 的逻辑完成“编译”：

  1. 能力表达：
    • 这里的汉字你会读吗？（提示：汉字 = 漢字 (かんじ)；读 = 読む）
    • 翻译：あなたは この 漢字 (かんじ) を 読むこと が できます か。
      You [this word reading] can?
  2. 兴趣表达：
    • 我的爱好是收集旧书。（提示：旧书 = 古 (ふる) い 本 (ほん)；收集 = 集 (あつ) める）
      私の趣味は古い本を集めることです。
      My interest [old book collecting] is.

  3. 时间顺序 (Before)：

    • 吃饭前，请洗手。（提示：洗手 = 手 (て) を 洗 (あら) います）
    • 游泳前，我做了热身运动。（提示：热身运动 = 準備運動 (じゅんびうんどう)；做 = しました）
    • 泳ぐ前に、私は準備運動をしました。

  4. 挑战题：

    • 我不会开车，所以每天「毎日：まいにち」坐巴士去大学。
    • (提示：开车 = 運転する；用 koto ga dekimasen；因为…所以… 用 kara 或 node)
      運転することができませんので、毎日バスで大学へ行きます。
      这个部分让你的日语表达瞬间提升了一个档次（从只会说动作，变成会描述能力和爱好）。加油！
