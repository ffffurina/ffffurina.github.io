### Lesson 18: 排序算法 —— 比较级与最高级

在英语中，比较级需要变形 (big $\rightarrow$ bigger) 或加词 (more expensive)。
在日语中，形容词**不需要任何变形**。我们只需要引入一个“参照物标记”：**より (yori)**。

#### **1. 比较级：A 比 B 更... (A > B)**

**核心逻辑**：
$$A \text{ は } B \text{ より } \text{ Adj } \text{ です}$$

* **より (yori)**：功能是**“Than” (比起...)**。
* **语序直译**：$A$ (Topic)，$B$ (Reference)，Adj。 $\rightarrow$ 至于 $A$ 嘛，比起 $B$ 来说，是 Adj 的。

**实战演练 (Implementation)**：

* **场景**：中国比日本大。
    * $A$: 中国 (ちゅうごく)
    * $B$: 日本 (にほん)
    * Adj: 大 (おお) きい
    * **Sentence**: **中国 (ちゅうごく) は 日本 (にほん) より 大 (おお) きい です。**

* **场景**：这辆车比那辆车贵。
    * **この 車 (くるま) は その 車 (くるま) より 高 (たか) い です。**

> **💡 进阶逻辑 (Inverse)**：
> 如果想说“A **不如** B”，我们通常用：
> $A$ **は** $B$ **ほど** Adj **否定形 (kunai)**。
> * **日本 (にほん) は 中国 (ちゅうごく) ほど 大 (おお) きくない です。**
> * (日本没有中国那么大。)
> * *注：`ほど ` 表示程度/Limit。*

#### **2. 二选一询问：哪个更...？ (Binary Choice)**

当你在两者之间做选择时（if A or B），不能用通用的 `どれ (dore)`，必须用专门的 **二选一疑问词**：**どちら (dochira)** (Which of the two)。

**提问公式**：
$$A \text{ と } B \text{ と } \text{ どちら が } \text{ Adj } \text{ ですか }$$
* **どちら**：**何方**
**回答公式**：
$$\text{Target } \text{ の 方（ほう）が } \text{ Adj } \text{ です }$$

* **の方(ほう)が：直译是“...的这一方 (The side of ...)”。

**实战演练**：

* **Q**: 足球和棒球，哪个更有趣？
    * **サッカー と 野球 (やきゅう) と どちら が 面白 (おもしろ) い ですか。**
* **A**: 足球更有趣。
    * **サッカー の ほうが 面白 (おもしろ) い です。**
面白い：(おもしろい)有趣的
#### **3. 最高级：在范围内...最... (Max Function)**

当选项超过 3 个，或者在某个群体中选出“最...”时，我们使用 **一番 (いちばん)**。

**公式**：
$$\text{Scope } \text{ で } \text{ Target } \text{ が } \text{ 一番 (いちばん) } \text{ Adj } \text{ です }$$

* **で**：表示**范围** (In the scope of)。
* **一番 (いちばん)**：Number One / Most。

**实战演练**：

* **场景**：日本料理中，寿司最好吃。
    * Scope: 日本料理 (にほんりょうり) **で**
    * Target: 寿司 (すし) **が**
    * Adj: **一番 (いちばん)** 美味 (おい) しい
    * **Sentence**: **日本料理 (にほんりょうり) で 寿司 (すし) が 一番 (いちばん) 美味 (おい) しい です。**

* **场景**：班级里谁最高？
    * **クラス で 誰 が 一番  背 (せ) が 高 い ですか。**
    * *(注：背が高い = 个子高)*

### 🛠️ 词汇穿插：反义形容词对 (Pairs)

为了练习比较，我们需要大量的“参数”。

| 汉字 (假名) | 含义 | 反义词 | 含义 |
| :--- | :--- | :--- | :--- |
| **速 (はや) い** | 快 | **遅 (おそ) い** | 慢 |
| **近 (ちか) い** | 近 | **遠 (とお) い** | 远 |
| **重 (おも) い** | 重 | **軽 (かる) い** | 轻 |
| **強 (つよ) い** | 强 | **弱 (よわ) い** | 弱 |
| **好 (す) き** | 喜欢 (Na) | **嫌 (きら) い** | 讨厌 (Na) |
| **上手 (じょうず)** | 擅长 (Na) | **下手 (へた)** | 笨拙 (Na) |

---

### 🧠 逻辑图解 (Logic Summary)

| 模式 | 助词/关键词 | 逻辑 |
| :--- | :--- | :--- |
| **A > B** | **A は B より** | A is [Adj] compared to B |
| **A < B** | **A は B ほど ...ない** | A is not as [Adj] as B |
| **Choice (2)** | **どちら** / **のほうが** | Which (of 2)? / The side of... |
| **Max (3+)** | **(Scope) で ... 一番** | In scope, X is Number 1 |

### ✅ 课后小练习 (Output)

请“编译”以下比较句：

1.  **基础比较 (A > B)**：
    * 飞机比新干线快。（提示：飞机 = 飛行機 (ひこうき)；新干线 = 新幹線 (しんかんせん)）
    * 飛行機は　新幹線より　速いです。
    * 今天比昨天热。（天气热：暑い;　物理上的热：熱　ねつ）
    * 今日は　昨日より　暑いです

2.  **二选一 (Binary)**：
    * **Q**: 肉和鱼，你更喜欢哪一个？（提示：喜欢是形容词 `suki`，助词用 `ga`）魚：さかな
    * **A**: 我更喜欢肉。
		肉と　魚と　どちら　が　好きですか。
		肉のほう　が　好きです。
3.  **最高级 (Max)**：
    * 一年之中，8月是最热的。（提示：一年之中 = 1年 (いちねん) で）
    * 一年で、八月が　一番暑いです
    * 在体育运动中，我最擅长网球。（提示：体育 = スポーツ；擅长 = 上手；网球 = テニス）
		スポーツで、テニスが　一番上手です。
